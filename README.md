## yy-react

我叫张越, 所以项目名为yy(越越)-react。源码主要是我在学习preact的源码后, 参考preact源码实现的。实现了JSX，组件，生命周期，diff，setState等核心功能。代码中有完善的中文注释, 对于了解虚拟DOM, diff算法, 还是有很大的帮助的。

```js

git clone https://github.com/BengBu-YueZhang/yy-react.git

npm install

npm start
```
